# 2022 창원시 빅데이터 공모전(기획)
2022 창원시에서 주최한 데이터 분석(기획, 분석)에서 기획 부문에 참여해 인구감소&노후주택 이라는 키워드를 가지고 진행한 데이터 분석이며 일부 코드를 업로드 한 것입니다.
특정 동네를 선정하여 노후 주택 > 빈집으로 될 공간들을 예측해 밀도가 높은 지점을 지도 위 표시하는 방법으로 진행 하였습니다.
수치적인 탐색을 넘어 현장 탐사를 통해 해당 장소가 가지고 있는 특유의 풍경, 거주하는 사람들 등등 데이터 밖의 요소들도 파악 하여 분석의 질을 높혔습니다.
이를 통해 도시정비 계획에 있어 객관적 선택 유도, 삶의 질 향상이라는 목표를 제시 최종 2위의 결과를 만들었습니다.
